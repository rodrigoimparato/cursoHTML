Cores podem ser nomeadas pelo nome em Inglês.
Cores podem ser noemadas por números hexadecimais na sequecia 1,2,3,4,5,6,7,8,9,a,b,c,d,e,f
Os códigos hexadecimais são mistura de cores RGB(Red-Green-Blue)

Como se lê um cor em hexadecimal?

#550000
Divide-se em 3 partes.
55 para Red -> uma porcentagem para vermelho
00 para Green -> nada para verde
00 para Blue -> nada para azul

Pode se simplificar o hexadecimal

Para partes repetidas pode se utulizada apenas 1.
Por Exemplo.
#ffffff pode virar #fff

Cores nomeadas em Iglês

Não se recomenda utilizar o nome da cor em iglês porque pode não funcionar em alguns browser
